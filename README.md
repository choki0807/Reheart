# ▌死而替生 — ReHeart

> *她们拿走了你的命，我把它还给你。但代价是，你要成为我。*

---

## ◈ 这是什么

一个关于身份剥夺与灵魂夺舍的交互式叙事游戏。

你是一个已经死去的女孩。一只猫将它的心脏塞进你冰冷的胸腔，让你以他人的身份继续行走。每一次夺舍，自画像上便有一抹色彩永远死去。当你剥落了所有颜色，你就成为了完美的防火墙组件。

**不再有情感。不再有色彩。**

---

## ◈ 游戏架构

```
序幕 → 标题 → 手术室 → 杀手 → 顶层公寓
  │      │      │        │        │
  │      │      │        │        └─ 终局：保留 or 碎裂
  │      │      │        └─ 激怒系统 / 武器订单
  │      │      └─ 身份审计（AI 驱动 · 3分制）
  │      └─ 点击初始化载体连接
  └─ 微信对话 · 自画像 · 猫的契约 · 手术台觉醒
```

| 幕 | 身份 | 核心机制 |
|:---:|:----:|:--------:|
| 序幕 | 自己 | 线性叙事 · 无交互 |
| 第二幕 | 外科医生 | AI 身份审计 · 3分制 |
| 第三幕 | 杀手 | 激怒系统 · 线索收集 |
| 第四幕 | 上层阶级之女 | 终局抉择 |

---

## ◈ 视觉美学

- **1px 极简主义** — 禁止圆角，禁止位图，所有视觉资产由 Inline SVG 构成
- **纯黑背景** — `#000000`，无妥协
- **灰度 UI** — `white/10` 至 `white/90`，配合 `backdrop-blur` 营造冰冷的果冻感
- **动态反馈** — 对话框律动 · 审计失败全屏噪点 · 枪击 Glitch

## ◈ 启动

```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入 OPENAI_API_KEY

# 开发模式
npm run dev

# 生产构建
npm run build
```
